# Certificates Folder

Add your certificate images here.

## Naming Convention
- Use descriptive names: `aws-certification.png`, `react-certificate.png`
- Supported formats: PNG, JPG, WEBP
- Recommended size: 1920x1080px or similar aspect ratio

## Current Certificates Needed
Based on `components/certificates-section.tsx`:

1. `aws-cert.png` - AWS Certified Solutions Architect
2. `fullstack-cert.png` - Full Stack Web Development
3. `ml-cert.png` - Machine Learning Specialization

## How to Add
1. Save your certificate image in this folder
2. Update the path in `components/certificates-section.tsx`
3. Restart the dev server to see changes

## Tips
- Scan certificates at high resolution
- Crop unnecessary whitespace
- Ensure text is readable
- Consider adding a subtle border or shadow
